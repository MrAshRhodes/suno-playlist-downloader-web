**3. Add counter function (after the `scrollToRow` function — find the closing `};` of `scrollToRow`, add after it):**
```typescript
const checkAndShowDonationModal = () => {
    const current = parseInt(localStorage.getItem('suno-download-count') || '0');
    const next = current + 1;
    localStorage.setItem('suno-download-count', String(next));
    if (next === 1 || next % 5 === 0) {
        setDonationModalOpen(true);
    }
};
```

Key details per D-01 and D-06:
- localStorage key: `suno-download-count` (verified not in use by any existing code)
- Formula: `next === 1 || next % 5 === 0` triggers on counts 1, 5, 10, 15, 20...
- Increment happens before the condition check so the counter always advances
- No permanent opt-out — modal re-appears per formula

**4. Add counter call at the very START of `downloadPlaylist` function (find `const downloadPlaylist = async () => {`, insert as the FIRST line inside the function body, BEFORE `setDownloadPercentage(0)`):**
