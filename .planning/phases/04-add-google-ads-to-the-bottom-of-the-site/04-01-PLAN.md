  <action>
Add the Google AdSense Auto Ads script tag to client/index.html inside the existing `<head>` element, after the Google Fonts stylesheet link and before the closing `</head>` tag.

Insert these exact lines after line 11 (the Google Fonts CSS link) and before line 12 (the closing head tag):

```html
    <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXX"
         crossorigin="anonymous"></script>
```

Replace `ca-pub-XXXXXXXXXX` with the real publisher ID provided in Task 1.

Per decisions:
- D-05: Auto Ads only — this single script tag is the entire implementation. Google determines ad placement. No manual `<ins class="adsbygoogle">` elements anywhere.
- D-06: Publisher ID hardcoded directly in the script src URL — no .env, no environment variable.
- D-07: `async` attribute on the script — standard AdSense practice, does not block page render.
- D-08: No page region exclusions — no `data-ad-region` attributes or `<div>` wrappers needed.
- D-09: Graceful ad-blocker degradation — the async script simply fails to load silently. No error handler, no detection script, no fallback message.
- D-10: No GDPR consent banner — Google Auto Ads uses its own built-in consent mechanisms.
- D-13: No "Advertisement" label — Auto Ads handles its own labeling per AdSense policies.

Do NOT add:
- Any `<ins class="adsbygoogle">` manual ad unit elements
- Any ad-blocker detection JavaScript
- Any consent/cookie banner code
- Any new CSS styles or React components
- Any `<meta>` tags for ads (Auto Ads does not require them)

The `crossorigin="anonymous"` attribute is part of the standard AdSense snippet and enables proper CORS handling for the external script.

The resulting index.html should look like:
```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <link rel="icon" type="image/png" href="/src/icons/icons8-playlist-96.png" />
    <title>Suno Playlist Downloader</title>
    <meta name="description" content="Download your Suno playlists - Web Version" />
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
    <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600&display=swap" rel="stylesheet">
    <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXX"
         crossorigin="anonymous"></script>
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/src/main.tsx"></script>
  </body>
</html>
```

(With the real publisher ID replacing ca-pub-XXXXXXXXXX)
